# RecipeBook

This is a simple Recipe book app built with Vue and Express. It uses plain json array to store recipes.

## Project management

https://www.pivotaltracker.com/n/projects/2634719

## Project setup

```
cp .env.example .env
```

```
npm install
```

### Running the app

```
npm start
```
